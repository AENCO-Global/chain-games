connect4 Class

Defined in: js/game.js:1

Connect4 game for 1-1 chat rooms in Chaatz. Users can start a game from Android/iOS clients from within a chatroom, make a move and transmit it to their chat partner via XMPP.

This game relies on the native chaatz client for communication and persistence of data.
